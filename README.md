# Smart Warehouse Case Study

This project was developed as part of a Smart Warehouse management module case study.

## Technologies

Backend
- .NET 9 Web API
- Entity Framework Core
- SQL Server
- Layered Architecture (Controller → Manager → Repository → Entity)

Frontend
- React
- TypeScript
- Vite
- Axios

## Features

- Company CRUD operations
- Multi-tenant filtering (companyId)
- Soft delete
- Server-side pagination
- Search / filtering
- Swagger API documentation
- React-based UI

## Backend Setup
cd SmartWarehouse_Project
dotnet run

Swagger:
https://localhost:xxxx/swagger

## Frontend Setup
cd smart-warehouse-ui
npm install
npm run dev

Frontend runs on:
http://localhost:5173


## Project Structure
smart-warehouse-case

├ smart-warehouse-ui # React Frontend
